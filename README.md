## 使用方式

### 网页版（GitHub Pages）

1. Fork 或上传到你的 GitHub 仓库
2. 进入仓库 Settings → Pages → Source 选择 main 分支
3. 几分钟后访问 `https://你的用户名.github.io/仓库名`

### PDF版导出

1. 用 Chrome 打开 `index.html`
2. `Ctrl/Cmd + P` → 另存为PDF
3. 建议设置：纸张A4、边距最小、背景图形勾选
